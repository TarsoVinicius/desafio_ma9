# Gerenciador de Produtos - Distribuidora Veterinária

Aplicação web para gerenciamento de produtos de uma veterinária, desenvolvida como desafio técnico.

## Tecnologias Utilizadas

- **Vue 3** + Composition API + `<script setup>`
- **TypeScript**
- **SCSS** para estilização
- **Pinia** (gerenciamento de estado)
- **Axios** + **json-server** (API REST simulada)
- **Vite** (build tool)
